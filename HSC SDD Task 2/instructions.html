Intructions

Access the Main Menu: Navigate to the main menu of the Array Finder application.
View the Array Table: Upon accessing the main menu, you will see an array table displayed on the screen. This table contains the initial array of numbers.
Edit the Array (Optional): If you wish to edit the array, click on the "Edit Array" button located on the right side of the table. This will enable you to input your own elements into the array.
Find Index of a Number: Enter a number into the input field labeled "Enter a number" located below the array table. Then, click the "Find Index" button to search for the index of the entered number within the array.
Sort the Array (Optional): If the array is not sorted and you encounter difficulties finding the index of a number, you can sort the array using the "Bubble Sort" button. This will rearrange the elements in ascending order.
Reset the Array (Optional): To revert the array back to its initial state, click the "Reset Array" button. This will restore the array to its original values.
Shuffle the Array (Optional): If you want to randomize the order of elements in the array, click the "Shuffle Array" button. This will shuffle the elements into a new random order.
View Result: After finding the index of a number or performing any other action, the result message area below the buttons will display the outcome of your operation, such as whether the number was found and at which index.
